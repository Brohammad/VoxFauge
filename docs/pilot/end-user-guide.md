# End-User Guide

How to use VoxForge as an operator or demo user — no engineering setup required.

## Try the public demo

1. Open https://voxforge.brohammad.tech/demo
2. Click **Run demo call**
3. Review the transcript, latency, and session ID

No account or API keys needed.

## Create an account

1. Open https://voxforge.brohammad.tech/dashboard
2. Register with your email and password
3. You are logged in automatically after registration

## Dashboard sections

| Section | What you can do |
|---------|-----------------|
| **Overview** | Session volume, outcome trends |
| **Knowledge Base** | Upload documents, test search |
| **Handoffs** | View pending human escalations |
| **Sessions** | Browse recent voice sessions |
| **Replay** | Inspect messages, tools, evaluations for one session |
| **Latency** | STT/LLM/TTS breakdown |
| **Evaluations** | Quality scores per turn |
| **Policy Presets** | Apply agent configuration bundles |

## Upload a knowledge document

1. Go to **Knowledge Base** in the sidebar
2. Select or create a collection
3. Choose a file (PDF, TXT, MD — not executables)
4. Wait for status `ready`, then use **Search Test**

## Review a session

1. Open **Sessions** → click a session ID
2. Or go to **Replay** and enter a session UUID
3. Review transcript, tool calls, and evaluation scores
