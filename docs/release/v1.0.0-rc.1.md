# VoxForge v1.0.0-rc.1

**Release date:** 2026-07-10  
**Live:** https://voxforge.brohammad.tech

## Highlights

- **Public production deployment** with HTTPS (Let's Encrypt), NGINX, Docker Compose
- **Product website** with interactive demo, screenshots, and documentation links
- **354+ automated tests** including 8 Playwright browser journeys
- **Voice AI platform**: unified pipeline, knowledge RAG, MCP tools, handoff, replay, evaluation
- **Operator dashboard** with SAML SSO, API keys, policy presets

## Quick start

```bash
git clone https://github.com/Brohammad/VoxForge.git
cd VoxForge
uv sync
docker compose up -d postgres redis
alembic upgrade head
uvicorn voxforge.main:app --app-dir src
```

Try the live demo: https://voxforge.brohammad.tech/demo
